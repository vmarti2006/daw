## sistema de edicion csv: 

``` 
	caracteristica_tabla,caracteristica_tabla,caracteristica_tabla
	contingut_taula,contingut_taula,contingut_taula
``` 
	
Las comas sirven para separar columnas i cada enter es una fila, sistema ultilzado para transportar de maner rapida la informacion.

---

## LLenguatje json: 

``` 
{
 "titul_taula":[
   {
     "titul_columna": "Contingut_fila_1",
   },
   {
     "titul_columna": "Contingut_fila_2",
   },
 ]
}
``` 

La llave{} sirve para abrir i cerrar texto, las comillas sirve para determinar texto, si no se gregan se ponen numeros, las comas sirven para finalizar un contenido i el corchete[] par indicar que pertenece a una tabla.

## LLenguatje markdown

El llenguatje markdown es un sistema de esciptura facil de entendre.
- Per dir els tituls sutilitza el hastack, cuants mes tingui mes vaix es el nivell del titul
- Per crea llistas utilitzem utilitzem els guions i per llistes numerades utilitzem numero seguit de un punt 
EX:  
  - hola
  - adios
  
  1. hola.
  2. adios
- Per agregar una imatge i url el que farem es el ![descripcio_del_contingut](url_o_direccio_imatge)
- Per agregar negreta i cursiva utilitzarem * per cursiva *posas una al principi i al final* mentres que **per la negreta poses dos**
